# R6 Stats Tracker

A command line application that looks up Rainbow Six Siege player stats using the R6Data API.

## Technologies Used
- C++
- libcurl
- nlohmann/json
- libpq (PostgreSQL)
- G++ Compiler

## Setup
1. Install PostgreSQL application
2. Setup pgAdmin server and database
3. Install libcurl, nlohmann, and libpq
4. Use cmake to link and build everything together
