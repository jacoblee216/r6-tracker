# R6 Stats Tracker

A command line application that looks up Rainbow Six Siege player stats using the R6Data API.

## Technologies Used
- C++
- libcurl
- nlohmann/json
- libpq (PostgreSQL)

## Setup
1. Install PostgreSQL
2. Install dependencies...
3. Update connection string in main.cpp

## Usage
1. Run the program
2. Enter your R6 username
3. Enter your platform (uplay, psn, xbl)
