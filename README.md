# R6 Stats Tracker

A command line application that looks up and saves R6 player stats using a R6 API.

## Technologies Used
- C++
- PostgreSQL
- G++ Compiler

## Libraries Used
- libcurl
- nlohmann/json
- libpq (PostgreSQL)

## Setup
1. Install PostgreSQL application
2. Setup pgAdmin server and database
3. Install libcurl, nlohmann, and libpq
4. Use cmake to link and build everything together
